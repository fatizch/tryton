Souscription
============

Il y a plusieurs façon de souscrire des contrats dans **Coog** (APIs, création
directe, etc.), nous allons ici nous concentrer sur la création manuelle
utilisant les processus paramétrés.

Du fait du lien avec le paramétrage de ces processus, ce qui est présenté ici
est un exemple relativement standard et peut différer ce qui est installé chez
un client donné. Toutefois, les grandes lignes resteront en général les mêmes,
d'autant que certaines actions sont nécessairement effectuées dans un certain
ordre (il ne sera jamais, malgré tout le paramétrage possible, de visualiser
l'échéancier tant que les informations des assurés / des garanties n'auront pas
été saisies).
